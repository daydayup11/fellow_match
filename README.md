# 伙伴匹配系统

基于 SpringBoot + Vue 的网站，支持用户管理，注册登录，随机智能匹配用户、支持寻找加入队伍，广场发布帖子，关注好友，查看共同关注好友，feed流推送动态等功能

## 后端技术栈：

1. SpringBoot
2. Mysql
3. Redis
4. Oss
5. MyBatis Plus
5. Swagger+Knife4j

## 部署步骤：
1. 将db文件夹下的user.sql导入数据库
2. 引入maven依赖
3. 修改application.yml里的mysql、redis、oss地址为自己的
4. 启动Application
