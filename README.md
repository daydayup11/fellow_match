# 伙伴匹配系统

基于 SpringBoot + Vue 的网站，支持用户管理，注册登录，随机智能匹配用户、支持寻找加入队伍，广场发布帖子，关注好友，查看共同关注好友，feed流推送动态等功能

技术栈：
1. SpringBoot
2. Mysql
3. Redis
4. Oss
5. Swagger+Knife4j
